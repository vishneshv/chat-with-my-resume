# Projects

## HireGraph
Agentic AI Interview Assistant that answers questions about Vishnesh's experience.

**Tech Stack:**
- LangGraph for agent orchestration
- FastAPI for backend API
- FAISS for vector search
- Groq for LLM inference
- Playwright for web scraping

**Features:**
- Intent classification for routing queries
- RAG-based knowledge retrieval
- GitHub integration for repo info
- Web search for external facts
- Session memory with SQLite
- Real-time SSE streaming

## Chatbot Builder (Kore.ai)
Low-code conversation flow designer for enterprise chatbots.

**Tech Stack:**
- Node.js backend
- Angular frontend
- MongoDB for storage
- Redis for caching
- Kafka for event streaming

**Impact:**
- Used by 100+ enterprise customers
- Reduced bot development time by 60%
